# 翻译插件适用于HBuilderX  
  
  本插件可实现代码编辑汉字自动替换英语,或英语自动替换汉字,或汉字替换成拼音,方便您添加注释,或起变量名称。增强您的代码的阅读性。
## 使用教程
> 由于hbuild暂未上架差价市场,暂时可手动加载插件。  
> 将该插件拷贝到HBuilderX的安装目录/plugins下,然后重启HBuilderX即可生效。  
  
> 右键>翻译模式 可更改模式  
> 右键>翻译源 可更改为谷歌,百度和有道,此模式仅适用于翻译的模式。
